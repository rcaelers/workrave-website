Workrave 1.8.3 released

by Raymond Penners in release

May 12, 2006

New Swedish and Catalan translations, updates to Czech translation. Workrave no longer consumes 100% CPU is if a break is started while the screen is locked. Fixed a bug that could cause breaks to stop running when ‘Suspend timer while inactive’ was disabled. Added support for storing configuration in a .ini file, and for configuring the data storage directory (this can be used to run Workrave from a memory stick). GNOME screen saver support added. The applet popup menu is now accessible when all timers hidden.

Workrave 1.8.2 released

by Raymond Penners in release

Jan 27, 2006

Version 1.8.2 mainly contains bug fixes: fixed a crash when using a third party gtk+ runtime, fixed a bug that caused status windows to be disabled when the micro-break timer was hidden, translation fixes and better hibernate/standby support on Unix.
