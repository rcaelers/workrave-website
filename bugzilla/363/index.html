Rob Caelers  Jan 28 2009 21:50:06 UTC
Bugs 363 and 741 talk about 'virtual desktops' (one monitor, multiple desktops) on Windows (virtual desktop have been standard on linux for years). Mouse/keyboard monitoring should (since 1.8.6) work on all desktops. However, break windows are only shown on the first desktop. This is why 741 is still open (363 only mentions monitoring).

805 mentions 'dual monitor' on Linux. I think this is quite different from 364 and 741.
