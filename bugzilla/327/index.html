Giles Weaver  May 8 2010 02:51:44 UTC
I've been following this bug for a while and am pleasantly surprised that the problem has gone away on upgrading to Ubuntu 10.04.
The bug appears to have been fixed in the Debian 1.9.1-1 release, which incorporated the new upstream 1.9.1 release.

More info here: http://packages.debian.org/changelogs/pool/main/w/workrave/workrave_1.9.1-3/changelog

The Workrave 1.9.1 release notes include the following entries:
# Fixed color in dark UI themes on Gnome.
# Workrave no longer shows two icons on GNOME notification panel
# Fixed tray icon tranparency 

I think this bug can probably be marked as resolved/fixed.
