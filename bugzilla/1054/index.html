Andrey Solis  Nov 23 2011 21:23:16 UTC
Created attachment 266
Workrave is still running.

I recently change my computer, then I downloaded and tried to run the 1.9.4 MSI (Windows) installer.  I have administration rights in the computer etc...
but...
The installer prompted me that i have workrave running!! (That's impossible!) and ask me to close it first, i have nothing to close, so i close the setup.
I downloaded the setup again but the installer always said that WorkRave was still running and that I needed to close it first...

I found a quick solution in:
http://sourceforge.net/mailarchive/message.php?msg_id=27667851

As Rob says: "It seems that if an explorer window is in the 'Program Files\Workrave' directory, that explorer window is also called 'Workrave'"

I was trying to install from a directory named Workrave, so as the above message said, i change the name of the directory to Workrave_Installer, and now the setup didn't ask me to close workrave, and now the setup is working good.
