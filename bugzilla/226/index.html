Comment 3
Kees-Jan Dijkzeul  Apr 9 2003 01:37:30 UTC
Tough call. There are good arguments either way.

Against remembering mode: When I start wr, I want it to be in normal mode,
because I want to be warned in time for my breaks, regardles of what mode I used
to be in.  Besides. The only marginally useful mode to remember is "quiet". If
the mode to be remembered is "suspended", then you shouldn't be starting wr at all.

In favour of remembering mode: I want to always start in mode "quiet". I'm only
interested in recording my behaviour. I do not want to be forced to break.

Disclaimer: These are not my opinions. I'm not sure whose they are, but they are
not mine :-)

Compromise between the two: New configuration option: Mode that wr should start
up in. Would that work for you, Sam?
Comment 4
Raymond Penners  Apr 9 2003 02:16:34 UTC
Oh no, that compromise is really bad: adding configuration options just because

http://www106.pair.com/rhp/free-software-ui.html some time, especially the
section on preferences.

IMHO, the mode should be remembered.
