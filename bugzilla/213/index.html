Rob Caelers  Mar 20 2003 12:29:06 UTC
Please re-read my comment above. There is no "C" in glib. Redhat does have this
pc file. I just checked. I's in glib2-devel-2.0.1-2.i386.rpm
(/usr/lib/pkgconfig/glib-2.0.pc)

To compile workrave you'll need at least glib 2.0, gtk+ 2.0 and gtkmm 2.0. As
far as I know, there is no gtkmm 2.0 package for Redhat 7. So you may have to
compile it yourself.

I don't think this bug describes a problem in workrave (except that it needs a
README containing all dependencies, badly). So I'm closing this bug. Feel free
to email me directly if you still have problems compiling workrave.
