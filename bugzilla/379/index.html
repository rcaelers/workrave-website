Raymond Penners  Apr 21 2004 07:54:59 UTC
User reports:

Regarding the exercise manager, I have a wish list for future versions: 
- possibility to select only some exercises (some might not be good for my
back or I don't like them). 
- possibility to add my own exercises (even if there's no animation, but
just some text and/or a picture). 
- random sequence. 
- possibility to have (one) exercice played during micro-pauses. 
- a sound every time the animation changes during one exercise (for example,
when your head changes side), so I can follow the rythm without looking.

Neal McBurnett  Aug 1 2006 02:22:15 UTC
A helpful start would be just noting that carefully editing the file:

 /usr/share/workrave/exercises/exercises.xml

(for the linux version) allows you to add new exercises or modify existing ones.  The changes usually seem to be picked up the next time
a rest is called for, though sometimes that seems not to happen.

So just giving some official hints about that on the web site (or here in bugzilla) would be handy.   Along with warnings about needing to back up any changes so they aren't wiped out during upgrades.....   Then at least we'd be getting more folks developing some alternative excercise patches.

Fixing the core dumps that happen when you get the xml wrong (e.g. leaving out the <sequence> element in exercises.xml) would be good also, but I know there are probably a lot of error cases there to consider....
