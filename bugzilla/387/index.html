Rob Caelers  Jan 28 2009 20:34:30 UTC
I'm closing this more than 4 year old bug because:

- I have never been able to reproduce it in past because I didn't have a tablet.
- I have not been able to reproduce it with version 1.9.0 and a Wacom Bamboo tablet.
- Workrave's activity monitor has changed a lot in the last 4 years. It has become less intrusive, It is very likely that the problem as you describe it is gone because of these changes.
- Workrave and gimp both use the Gtk+ library. The version used by Workrave 1.9.0 is much more recent than the one from 2004. This may also explain why I don't see the problem in 1.9.0

Feel free to reopen the bug if you still can reproduce the problem with a recent version of Workrave.
