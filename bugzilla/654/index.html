Comment 1
Ray Satiro  Jul 19 2007 03:20:15 UTC
Hello,

There is a timer issue in 1.8.4 but I don't think it's related to what you describe.

Are you sure you're using 1.8.4?
Did you install Workrave? Do you remember choosing minimal or full install?
Do you suspend Workrave, or does it suspend itself?
Is Workrave using the English language?
Is networking enabled? Please check.
Do you have any anti-virus or anti-hook software installed?

If Workrave is operating in suspended mode the activity monitor is suspended. And if you also have "Suspend timer when inactive" enabled, the timers will stop and eventually reset because the activity monitor is suspended.
