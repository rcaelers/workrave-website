Ray Satiro  Jan 16 2009 05:57:04 UTC
>But the the dialog still shows in MS Word window and makes it unusable,

Do you recall if the rest break timer was active after you postponed? Unusable how? Did the application respond in any way?

The only known scenario I can think of where this could possibly happen is when the application behind the rest break window has stopped responding / crashed (unrelated to Workrave). In this scenario if the rest break window closes, manual or otherwise, and the app behind it is not responding, the app is not able to repaint its area. So you'll see the rest break window even though it is already gone.

If this ever happens to you again please take some screenshots.

Thanks
