Description
Sec  Dec 14 2005 10:37:35 UTC
I like workrave very much, but when playing a game in fullscreen it is very annoying if workrave pops up its break warnings. I would like to have an (optional?) Hotkey to disable workrave. Re-activation coult either be another hotkey, or it could re-enable itself autmatically after X minutes of inactivity.
Comment 1
Stefan Wrobel  Feb 9 2006 23:53:31 UTC
I agree, I'd like to see this feature added because I use Workrave on my machine at work, and sometimes people come by to talk to me about something and I'm showing them code and up pops the micro-break window, and then I have to fight it off the screen.  I can go to the tray icon and right-click and suspend it, but it would be nice if I could just have a hotkey bound to suspend/un-suspend it, like ctrl+alt+s or something like that.
