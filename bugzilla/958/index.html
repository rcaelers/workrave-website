Hi =) Thank's for this software ;)

Firstly and more important, it would be great to be able to customize the Inactive Timer when we are AFK, because I usually do a lot of reading and so keep without doing anything on keyboard nor mouse for about 20 seconds, but the inactive timer freezes the rest timers after only 3 sec, so I need to deactive it and its not rare that I go AFK and when I come back I have a bonus rest break :p

...

+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

Great software.  However, I have used version 1.9.0 for some time and when I decided to upgrade to the latest version found that the "Suspend timer when inactive" checkbox is no longer available.  (It is very helpful for anyone who reads a lot on the computer because unchecking it allows the timers to keep running while reading.  Hence, you still get your breaks as scheduled, rather than having the timer reset.)  Consequently, I have reinstalled version 1.9.0.  Please reintegrate this feature into WorkRave.

Cheers,
Jim

Kees-Jan Dijkzeul  Jan 5 2011 17:26:00 UTC
(In reply to comment #0)

> Great software.  However, I have used version 1.9.0 for some time and when I
> decided to upgrade to the latest version found that the "Suspend timer when
> inactive" checkbox is no longer available. 

Correct. It got relabeled. When you right click on the workrave window or the applet, you'll get the menu. It has a new menu item "reading mode". If you activate it, timers will not be suspended when you are inactive, just like you are used to in 1.9.0

The idea is that it now is much easier to switch reading mode on and off (used to be three checkboxes hidden deep down in the preferences dialog), such that you can more easily adapt workrave to your activities.
