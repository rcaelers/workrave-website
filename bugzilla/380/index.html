Rob Caelers  Apr 27 2004 12:11:44 UTC
The bonobo errors are probably caused because workrave cannot find the .server
files. They are installed in /usr/local/lib/bonobo/servers/. Gnome might not be
looking in this directory...

When workrave is running, try: 
  bonobo-activation-run-query "_active == true" | grep Workrave

It should output:
  OAFIID:GNOME_Workrave_Factory
  OAFIID:GNOME_Workrave_WorkraveControl

About the crash: well spotted. I think it is already fixed in CVS. Could you try
a nightly build. 1.6.0 will be release real soon (I hope)..
